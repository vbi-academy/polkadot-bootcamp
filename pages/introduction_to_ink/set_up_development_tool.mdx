# Cài đặt môi trường làm việc

## Cài đặt rust 
1. Bạn có thể xem phần [Cài đặt môi trường Substrate](../install_and_run_substrate_node/install_development_tool.mdx) để cài đặt Rust 

2. Cài thêm C++ compiler 
```bash
rustup component add rust-src
```

3. Kiểm tra Rust đã cài đặt ở máy 

```bash
rustup show
```


## Cài đặt CLI 
Gồm tạo, buid, deploy ink! smart contract thông qua cli. Có thể sử dụng `ink! cli` hoặc `pop cli` 


### Ink! CLI
```
cargo install --force --locked cargo-contract
```

### POP CLI
```bash
cargo install --force --locked pop-cli
```
